# 10web.io Style Clone

This project is a demo website that closely matches the design, color scheme, UI, and UX of [10web.io](https://10web.io/). All graphics and content are placeholdersâ€”you can change them as needed.

## Tech Stack
- React (with Vite)
- Tailwind CSS

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:5173](http://localhost:5173) to view the site.

## Customization
- All content and images are placeholders. Replace them in the `src/` directory as needed.
- Tailwind CSS is used for styling. You can customize the theme in `tailwind.config.js`.
